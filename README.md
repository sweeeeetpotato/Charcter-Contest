# <천하제일 캐릭터 만들기 대회>

### 캐릭터 소개
* 9시부터 18시까지 수업을 듣다보니 하루의 반이 순식간에 사라지고, 수업이 끝난 후에는 복습하고 과제하느라 바쁜 일상을 보내고 있습니다. 
* 과제할 시간이 부족할 정도로 시간이 너무 빨리가는 요즘을 느끼며, 시간이 빨리가서 슬퍼하는 시계 캐릭터를 만들어보았습니다.
